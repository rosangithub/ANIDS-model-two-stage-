Anomaly-Based Network Intrusion Detection Using Two-Stage Machine Learning is a machine learning-driven security solution designed to detect and classify malicious network activities in real time. Using the CICIDS2017 Dataset, the system follows a two-stage architecture: the first stage detects whether traffic is benign or malicious, and the second stage classifies the specific attack category. This approach enhances detection accuracy while maintaining efficient real-time performance.

üéØ Project Objectives

Main Objective:

  • To develop a real-time anomaly-based Network Intrusion Detection System using a two-stage machine learning approach.

Specific Objectives:

  • To preprocess the CICIDS2017 dataset for model training.
  • To select important network traffic features.
  • To develop a Stage-1 Random Forest model to detect normal and malicious traffic.
  • To develop a Stage-2 CatBoost model to classify different attack types.
  • To implement real-time packet capture using Scapy and Npcap.
  • To deploy the system using Flask with a web-based interface.
